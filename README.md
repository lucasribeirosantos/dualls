# DuaLLs — Landing Page

> Automação que conversa. Design que conecta.

Landing page de vendas da **DuaLLs** — automação de atendimento com IA + Webdesign em uma solução só.

## Stack

Site estático, sem build. Apenas HTML, CSS e JavaScript puro.

| Arquivo | Descrição |
|---|---|
| `index.html` | Estrutura e conteúdo da página |
| `styles.css` | Estilos, identidade visual e animações |
| `script.js` | Preloader, animações de scroll, calculadora de economia |
| `logo-icon.png` | Ícone da marca (cabeçalho, favicon, preloader) |
| `logo-escrita.png` | Logotipo completo (rodapé) |
| `hero-img.png` | Imagem da hero (estátuas) |
| `og-image.png` | Imagem de compartilhamento (Open Graph) |
| `robots.txt` · `sitemap.xml` · `site.webmanifest` | SEO / AEO |

## Rodar localmente

Basta abrir o `index.html` no navegador. Para um servidor local:

```bash
npx serve .
```

## Deploy

Site 100% estático. É só enviar os arquivos da raiz para a pasta pública da hospedagem
(`public_html` na Hostinger). O ponto de entrada é o `index.html`.
